# nBase2 Systems TIDL (Texas Instruments Deep Learning) EdgeAI

This repo contains the steps followed in developing the model, compiling it for the tidl and inferencing it in the AM62A target.

## Installation

1. Clone the repository
```bash 
git clone https://github.com/jaigurul-nbase2/tf_lite_tidl.git 
cd tf_lite_tidl
```

2. For model development move to the model_development directory
```bash 
cd model_development
```

3. For compilation of the developed model move to compile_model/compile directory
```bash 
cd compile_model/compile
```
